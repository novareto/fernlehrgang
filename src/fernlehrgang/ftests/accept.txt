Do a functional doctest test on the app.
========================================

Let's first create an instance of Fernlehrgang at the top level:

   >>> from fernlehrgang.app import FernlehrgangApp
   >>> root = {} 
   >>> root['app'] = app = FernlehrgangApp()
   >>> import transaction; transaction.commit()

   >>> from fernlehrgang.interfaces.app import IFernlehrgangApp
   >>> IFernlehrgangApp.providedBy(app)
   True

Run tests in the testbrowser
----------------------------

Create a browser and visit the instance you just created:

   >>> from megrok.nozodb.utils import config, NOZODBWSGIPublisherApplication
   >>> wsgi_app = NOZODBWSGIPublisherApplication(handle_errors = True)
   >>> wsgi_app
   <megrok.nozodb.utils.NOZODBWSGIPublisherApplication object at ...>

   >>> from zope.testbrowser.wsgi import Browser
   >>> browser = Browser(wsgi_app=wsgi_app)
   >>> browser.open('http://localhost/app')
   >>> browser.headers['Status'].upper()
   '200 OK'
